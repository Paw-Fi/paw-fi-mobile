# Suggested Commands for Moneko Mobile

## Development
*   **Run App**: `flutter run`
*   **Run with flavor (if applicable)**: `flutter run --flavor dev` (check schemes)

## Code Generation
*   **Run Build Runner**: `dart run build_runner build --delete-conflicting-outputs`
*   **Watch Build Runner**: `dart run build_runner watch --delete-conflicting-outputs`

## Quality Assurance
*   **Linting**: `flutter analyze`
*   **Testing**: `flutter test`

## Dependency Management
*   **Get Dependencies**: `flutter pub get`
*   **Upgrade Dependencies**: `flutter pub upgrade`
